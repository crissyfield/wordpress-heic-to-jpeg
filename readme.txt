== Description ==

Convert HEIC images to JPEG format when upload to the Media Library.

This requires ImageMagick 7.0.8-46 or above to be installed on the server.

== Installation ==

1. Install **HEIC to JPEG** either via the WordPress.org plugin repository or by uploading the files to your
   server. (See instructions on [how to install a WordPress
   plugin](http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/)).
2. Activate **HEIC to JPEG** on the Plugins page.
